# **App Name**: Meeting Automator

## Core Features:

- Obtención de disponibilidad: Obtener los horarios no disponibles de la API proporcionada para bloquear esos espacios en el calendario de reservas.
- Visualización del calendario: Mostrar las fechas disponibles en un calendario, resaltando los días reservables.
- Selección de franja horaria: Enumerar intervalos de 30 minutos (09:00-18:00) o 60 minutos según el tipo de reunión seleccionado, para el día seleccionado, excluyendo los horarios de la API de no disponibilidad.
- Creación de reunión (Google Meet): Crear una reunión de Google Meet a través de la API de Google Calendar cuando se selecciona una franja horaria.
- Registro de reserva: Registrar la nueva reunión en la API de programación para crear la reunión y almacenar sus datos.
- Envío de invitación: Enviar un correo electrónico de confirmación al visitante con los detalles de la reunión (fecha, hora, enlace de Meet).
- Sugerencia de tiempo inteligente: Sugerir horarios óptimos para ambas partes utilizando IA, teniendo en cuenta las diferencias de zona horaria, los descansos comunes y las duraciones típicas de las reuniones, mejorando la experiencia de programación al minimizar la comunicación de ida y vuelta, actuando como una herramienta de asistente de programación.

## Style Guidelines:

- Color primario: Azul tranquilo (#64B5F6) para confianza y fiabilidad.
- Color de fondo: Azul muy claro (#E3F2FD), casi blanco, para proporcionar un fondo limpio y sin distracciones para las actividades de programación.
- Color de acento: Naranja energético (#FFB74D) para resaltar los espacios disponibles y la llamada a la acción 'Programar', atrayendo la atención del usuario.
- Emparejamiento de fuentes: 'Poppins' (sans-serif) para titulares y descripciones cortas para ofrecer una sensación moderna y geométrica. 'PT Sans' (sans-serif) se utiliza para el texto del cuerpo para mejorar la legibilidad y proporcionar un toque humanista.
- Utilice iconos limpios y minimalistas para la navegación del calendario y las pantallas de confirmación.
- Mantenga un diseño limpio y receptivo con una jerarquía visual clara, asegurando una fácil navegación en todos los dispositivos.
- Utilice animaciones sutiles para confirmar las reservas y guiar a los usuarios a través del proceso de programación.
